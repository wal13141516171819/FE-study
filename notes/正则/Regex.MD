# 正则
## 匹配写法
``` TypeScript
str.match(reg); 
```
## 匹配规则
|模式|规则|说明|
|:-|:-|:-|
|匹配单个任意字符|通过.a进行匹配|可以匹配sa、aa等任意字符|
|匹配单个指定字符|通过[ab]a进行匹配|可以匹配aa、ab任一字符|
|匹配单个字符区间内字符|通过a[0-9]|可以匹配a0-a9任一字符|
|字符区间排除字符^|通过[^ab]s|可以匹配除as、bs之外的以s结尾的连个字母字符|
|\d|等价于[0-9]||
|\D|等价于[^0-9]||
|\w|匹配一个字母、数字和下划线||
|\W|匹配一个非字母、数字和下划线||
|\s|匹配除[\b]之外的空白字符||   
|\S|反向\s||
|\x|匹配16进制数字的前缀|
|\0|匹配8进制数字的前缀||
